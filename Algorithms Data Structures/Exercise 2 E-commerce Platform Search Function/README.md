# Exercise 2 - E-commerce Platform Search Function

## Objective
Implement Linear Search and Binary Search for product searching.

## Algorithms Used
- Linear Search
- Binary Search

## Time Complexity

Linear Search
- Best Case : O(1)
- Average Case : O(n)
- Worst Case : O(n)

Binary Search
- Best Case : O(1)
- Average Case : O(log n)
- Worst Case : O(log n)

## Conclusion
Binary Search is faster than Linear Search but requires the array to be sorted.
