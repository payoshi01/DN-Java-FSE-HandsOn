# Exercise 7 - Financial Forecasting

## Objective
Predict future financial value using recursion.

## Algorithm
Recursive Algorithm

## Input
Current Value = 10000

Growth Rate = 10%

Years = 5

## Output
Future Value after 5 years = 16105.10

## Time Complexity
O(n)

## Space Complexity
O(n)

## Optimization
Use an iterative solution to reduce space complexity to O(1).
