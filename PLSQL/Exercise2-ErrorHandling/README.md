# Exercise 2 - Error Handling

## Scenarios

1. SafeTransferFunds
   - Transfers money between accounts.
   - Rolls back the transaction if an error occurs.

2. UpdateSalary
   - Updates an employee's salary.
   - Handles invalid employee IDs.

3. AddNewCustomer
   - Adds a new customer.
   - Prevents duplicate customer IDs.

## Concepts Used
- Stored Procedures
- Exception Handling
- COMMIT
- ROLLBACK
- RAISE_APPLICATION_ERROR
